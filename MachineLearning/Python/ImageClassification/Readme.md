Image Classification (Python 3.9.25)

Data Set: https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia?select=chest_xray
This dataset contains 5,863 chest X-ray images categorized into two classes: Normal and Pneumonia (Bacterial or Viral). The images are organized into separate folders for training and testing.
